Aprofundamento do Projeto

Objetivo

A ideia central do projeto foi criar uma cafeteira capaz de fazer praticamente tudo sozinha: moer os grãos, preparar o café e entregar a bebida pronta sem que alguém precise ficar monitorando. Queríamos mostrar como a automação pode facilitar até tarefas simples do dia a dia.

Metodologia

Para isso, começamos escolhendo os componentes certos e entendendo como cada parte funcionaria no sistema. Depois fizemos a montagem física, instalamos os servo motores e começamos a programar o Arduino para coordenar cada etapa do preparo.

A programação foi a parte mais detalhada, porque cada motor precisava agir na hora certa: liberar os grãos, moer, mover o pó para o filtro e controlar a água quente. Fizemos vários testes ajustando tempos, movimentos e quantidades até tudo funcionar de forma fluida.

Resultados e Discussão

O protótipo conseguiu realizar todas as etapas sozinho, exatamente como planejado. Durante os testes, percebemos pequenos detalhes que precisavam de ajustes — principalmente a velocidade dos motores e a quantidade de grãos liberada — mas nada que impedisse o funcionamento.

Mesmo ainda sendo um protótipo, já deu para ver que o sistema tem bastante potencial. Inclusive, abre espaço para melhorias futuras, como sensores extras, controle por aplicativo ou até integração com assistentes virtuais.

Conclusão

O projeto mostrou na prática como eletrônica e programação podem trabalhar juntas para automatizar uma tarefa simples como fazer café. Além de funcionar bem, ele ajudou a entender todo o processo por trás de um equipamento automatizado, mostrando como pequenas etapas precisam conversar entre si para que tudo aconteça do jeito certo.
