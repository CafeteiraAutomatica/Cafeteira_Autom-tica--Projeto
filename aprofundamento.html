Aprofundamento do Projeto

A ideia central do projeto foi criar uma cafeteira capaz de realizar praticamente todas as etapas sozinha: moer os grãos, preparar o café e entregar a bebida pronta sem que alguém precise monitorar o processo. O objetivo era mostrar como a automação pode facilitar tarefas do dia a dia.

Para isso, foram selecionados os componentes adequados e entendido como cada parte funcionaria dentro do sistema. Em seguida, foi realizada a montagem física, instalaram-se os servo motores e programou-se o Arduino para coordenar cada etapa.

A programação foi a parte mais detalhada, pois cada motor precisava agir no momento exato: liberar os grãos, moer, mover o pó para o filtro e controlar a água quente. Foram feitos diversos testes, ajustando tempos, movimentos e quantidades, até garantir que tudo funcionasse de forma fluida.

O protótipo conseguiu executar todas as etapas sozinho, exatamente como planejado. Durante os testes, foiram identificados pequenos ajustes necessários (principalmente na velocidade dos motores e na quantidade de grãos liberada), mas nada que comprometesse o funcionamento.

Mesmo sendo um protótipo, o sistema demonstrou grande potencial. Há espaço para melhorias futuras, como sensores adicionais, controle via aplicativo ou integração com assistentes virtuais.

O projeto evidenciou como eletrônica e programação podem trabalhar juntas para automatizar tarefas simples, como fazer café. Além de funcionar bem, permitiu compreender o processo completo por trás de um equipamento automatizado, mostrando como cada etapa precisa se comunicar corretamente para que tudo aconteça conforme planejado.
