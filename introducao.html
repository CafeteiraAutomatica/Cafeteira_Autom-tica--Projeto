Introdução ao Projeto

Este projeto apresenta o desenvolvimento de uma cafeteira automática com moedor integrado, capaz de moer os grãos e preparar café fresco de forma totalmente autônoma. A proposta combina eletrônica, programação e automação para demonstrar, na prática, como diferentes componentes podem trabalhar juntos para realizar uma tarefa cotidiana.

Objetivo Geral

Construir um sistema automatizado que entregue café pronto sem intervenção manual, utilizando Arduino, sensores e servo motores para controlar cada etapa do processo.

O que o sistema faz

  • Move os grãos até o moedor;
  • Aciona o mecanismo de moagem;
  • Transfere o pó para o recipiente;
  • Controla a liberação de água quente;
  • Coordena todas as ações por programação.

Produzido por alunos do IFSP.
